Columbia University
Principal Investigator: Andrea Califano, Ph.D.

B Cell Interactome
Contact: Celine Lefebvre, Ph.D., Columbia University
Web Link: http://amdec-bioinfo.cu-genome.org/html/BCellInteractome.html

The B cell interactome (BCI) is a network of protein-protein, protein-DNA and modulatory interactions in human B cells. The network contains known interactions (reported in public databases) and predicted interactions by a Bayesian evidence integration framework which integrates a variety of generic and context specific experimental clues about protein-protein and protein-DNA interactions - such as a large collection of B cell expression profiles - with inferences from different reverse engineering algorithms, such as GeneWays and ARACNE. Modulatory interactions are predicted by MINDY, an algorithm for the prediction of modulators of transcriptional interactions.

TransfactomeDB
Contact: Harmen Bussemaker, Ph.D., Columbia University
Web Link: http://bussemaker.bio.columbia.edu:8080/YeastTransfactomeDB/

Accurate and comprehensive information about the nucleotide sequence specificity of trans-acting factors (TFs) is essential for computational and experimental analyses of gene regulatory networks. The Yeast Transfactome Database is a repository of sequence specificity models and condition-specific regulatory activities for a large number of DNA- and RNA-binding proteins in Saccharomyces cerevisiae. The sequence specificities in TransfactomeDB, represented as position-specific affinity matrices (PSAMs), are directly estimated from genomewide measurements of TF-binding using our previously published MatrixREDUCE algorithm, which is based on a biophysical model. For each mRNA expression profile in the NCBI Gene Expression Omnibus, we used sequence-based regression analysis to estimate the post-translational regulatory activity of each TF for which a PSAM is available. The trans-factor activity profiles across multiple experiments available in TransfactomeDB allow the user to explore potential regulatory roles of hundreds of TFs in any of thousands of microarray experiments.

PREPPI
Contact: Barry Honig, PhD, Columbia University (honig_software@c2b2.columbia.edu)
Web Link: http://bhapp.c2b2.columbia.edu/PREPPI/

PREPPI is a database of predicted and experimentally determined protein-protein interactions (PPIs) for yeast and human. Predicted interactions are assigned a likelihood using a Bayesian framework that combines structural, functional, evolutionary and expression information. The database contains ~2 million predictions including 31,402 for yeast and 317,813 for human that are considered high confidence based on our analysis. Experimentally determined interactions are compiled from a set of public databases (e.g., DIP, IntAct, HPRD, etc).

PTMScout
Contact: ptmscout_admin@mit.edu
Web Link: http://ptmscout.mit.edu

PTMScout is a tool for viewing and analyzing high-throughput post-translational modification proteomic data, with a particular eye towards novel hypothesis generation. External data from Gene Ontology, PFAM, and Scansite are integrated with user data automatically, as well as expression data in a large number of tissue types and cell lines. Additionally, all post-translational proteomic mass spectrometry datasets from the White Lab at MIT are banked here, and other users may include their data on request.

Oregon Health & Science University
Principal Investigator: Joe Gray, Ph.D.

Contact: Laura Heiser, Ph.D. (heiserl@ohsu.edu)

The OHSU ICBP has generated a substantial profiling compendium on a panel of ~50 breast cancer cell lines. Cell lines in the collection were selected to mirror the phenotypic, genomic, and expression heterogeneity of human breast cancers as well as important recurrent molecular features found in primary tumors. The lines were characterized in extensive molecular detail in order to allow identification of molecular features most strongly associated with biological response. Molecular features measured include genome copy number (SNP6.0 arrays), gene expression (U133A, Exon Array, RNAseq), alternative splicing (exon array, RNAseq), mutation detection (Exome seq), miRNA expression, gene promoter methylation (Illumina Infinium Kit), cytokine secretion, protein expression (Western blots and RPPA arrays), and drug response (>100 compounds). Additionally, we have conducted two detailed studies of the dynamics of drug response, in which a select set of cell lines were treated with up to 4 therapeutic compounds, and then profiled at 8 time-points to assess changes in protein expression, as measured by RPPA array.

Sage Bionetworks
Principal Investigator: Stephen Friend, M.D., Ph.D.

SYNAPSE
Conatct: synapseInfo@sagebase.org
Web Link: https://synapse.sagebase.org/ - Home:null

Synapse is an innovation compute space that brings together scientific data, tools, and disease models into a Commons that enables true collaborative research for those who want to share and analyze data together. The platform consists of a web portal, web services, and integrations with data analysis tool and is organized around novel “Analysis Communities” that any scientist can create or join to work on complex medical problems. In Synapse you can:

  • Search across public data including curated GEO, TCGA, and Array Express
  • Find raw data or follow an analysis path to processed results
  • Load data directly into analytical tools like R, and record your analysis
  • Create Public or Private projects to share your work

Synapse is designed to enable many layers of collaboration: within a single team, between research teams, between research teams and industry partners, and within pharmaceutical companies.

Examples of datasets in Synapse:

Synapse Commons Repository
Conatct: synapseInfo@sagebase.org
Web Link: https://synapse.sagebase.org/ - Synapse:syn150935

The Synapse Commons Repository is a convenient repository of raw genomic data and corresponding phenotypic information from publicly available datasets, available with no restrictions on use. The data are from public repositories like NCBI GEO, ArrayExpress from EBI, or TCGA, as well as any source that can be accessed through the web, including a URL made available through an individual lab, a core facility, or a cloud based storage site like an Amazon S3 bucket.

Sage / DREAM breast cancer prognosis challenge
Web Link: https://synapse.sagebase.org/#BCCOverview:0

The Sage / DREAM Breast Cancer Prognosis Challenge, announced in April 2012 and concluding at the end of this year, is intended to assess the accuracy of predictive computational models designed to predict breast cancer survival. The computational models are built using both the clinical information about the patient's tumor as well as genome-wide molecular profiling data including gene expression and copy number profiles. The Challenge is fueled by the generous donations of two breast cancer clinical study data sets (Metabric and Oslo Val) obtained by Carlos Caldas of Cancer Research UK and Anne-Lise Borresen-Dale of Oslo University Hospital.

Hormone Receptor Target Binding Loci Database (HRTBLDB)
Contact: Victor Jin, Ph.D. (Victor.Jin@osumc.edu)
Web Link: http://motif.bmi.ohio-state.edu/hrtbldb/

Data from high-throughput lab techniques (ChIP-chip, ChIP-Seq, and ChIP-PET) are integrated in to this unified database to aid research on the hormone signaling pathways that regulate gene expression. Hormone induced differential gene expression has been implicated in numerous pathological conditions. Given this implication, research in hormone receptor binding sites is increasing at a rapid pace, and with the advent of whole genome analysis, the volume of data generated by this research is immense. HRTBLDb is built to help researchers collaborate and prosper from this explosion of data.

Endometrial Cancer Patient Methylation Database (ECPMDb)
Contact: Victor Jin, Ph.D. (Victor.Jin@osumc.edu)
Web Link: http://motif.bmi.ohio-state.edu/brcaChileV2/

The Endometrial Cancer Patient Methylation Database (ECPMDb) consists of whole genome methylation data in 92 endometrial cancer patients and 12 normal tissues conducted by the MIRA-seq protocol. The ECPMDb makes rich, interactive genomic visualizations of the hyper-methylated regions across the human genome and can be accessible over the internet and be shared with collaborators. The graphical rendering of the data is done in a dynamic, modular fashion, using Scalable Vector Graphics that are rendered in the browser by Javascript. The SVG renderer is run by a custom Javascript program using the jQuery and Raphael Javascript libraries. The data is supplied to the client in Javascript Object Notation from the server, which is then used by the program to dynamically render both the SVG graphical visualization and the HTML tabular representation of the data being visualized. The primary advantage of this database visualization is its rich and interactive data display. Moving the cursor over an element in the browser will display its coordinates, the GC content at that site, or other information as assigned.

miR2Disease
Contact: Yunlong Liu, Ph.D. (yunliu@iupui.edu)
Web Link: http://www.mir2disease.org/

miR2Disease, a manually curated database, aims at providing a comprehensive resource of miRNA deregulation in various human diseases. Each entry in the miR2Disease contains detailed information on a miRNA-disease relationship, including miRNA ID, disease name, a brief description of the miRNA-disease relationship, miRNA expression pattern in the disease state, detection method for miRNA expression, experimentally verified miRNA target gene(s), and literature reference.
